unimod.py - This is the script that was given during the Unimod challenge that was used for the encryption of the flag.

out - This file contains the ciphertext of the encrypted flag for the user to decrypt and retrieve.

reverse.py - This script can be used to brute force the encryption and print the flag to the screen.

getCharIntVal.py - This script gives useful information such as the highest and lowest Unicode character values of the ciphertext, it was built upon and used to shift the ciphertext by the proper value and print the flag to the screen.
